SUKI_ai_skin_analysis_dispenser
SU.KI (Sun Keep In-check) is an AI-powered sunscreen dispensing system that performs real-time skin analysis using computer vision and provides personalized sunscreen recommendations before dispensing.
The system emphasizes user safety, consent, and hygiene, making it suitable for deployment in public spaces.

Objectives
  Analyze skin tone using live camera input
  Classify skin type (Dry / Normal / Oily)
  Recommend suitable SPF and sunscreen brands
  Enable user-consent-based scanning (camera activates only after user action)
  Simulate an automated sunscreen dispensing mechanism

Technologies Used
  Programming: Python, JavaScript
  Framework: Flask
  Computer Vision: OpenCV
  Frontend: HTML, CSS
  Platform: Windows
  Hardware-ready: Arduino / ESP32 (future integration)

How to run:
  pip install flask opencv-python numpy
  python app.py
